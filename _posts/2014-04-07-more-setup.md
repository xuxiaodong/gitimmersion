**目的**

获得教程材料，并准备执行。

### 获得教程包

从以下地址获得 Git 教程包：

<http://gitimmersion.com/git_tutorial.zip>

### 解包

教程包有一个主目录“git\_tutorial”及三个子目录：

* html：HTML 文件。让你的浏览器打开 html/index.html。
* work：空的工作目录。你可以在这里创建仓库。
* repos：预先打包的 Git 仓库，这样你可以在教程中的任何地
  方跳转。如果你遇到问题，那么只需复制想要的实验到你的工
  作目录。
