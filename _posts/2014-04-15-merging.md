**目的**

学习如何合并两个分叉的分支以便将更改带回到单一分支。

### 合并分支

合并将两个分支中的更改结合在一起。让我们回到 greet 分支，
并将 master 合并过来。

```
$ git checkout greet
$ git merge master
$ git hist --all
```

```
$ git checkout greet

$ git merge master
```

通过周期性地合并 master 到 greet 分支，你可以选择 master 的
任意更改，并保持 greet 中的更改与主干中的更改兼容。

然而，这会产生丑陋的提交图。稍后我们将看看变基替代合
并这个选项。
